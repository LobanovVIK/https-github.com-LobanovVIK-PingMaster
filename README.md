PingMaster 🚀
A lightweight network monitoring tool for iOS.

PingMaster allows you to track the availability of multiple hosts in real-time. Built with SwiftUI and designed for stability, it features a clean MVVM architecture and full backward compatibility with iOS 15.0+ (perfect for keeping older devices like the iPhone 7 Plus useful).

Key Features:
Real-time Status: Instant ICMP ping monitoring.

Batch Actions: Start or stop all pings with a single tap.

Retro-Compatibility: Optimized for iOS 15.6 and older hardware.

MVVM Design: Clean, maintainable, and modern Swift code.

PingMaster 🚀
PingMaster — это легкое и эффективное iOS-приложение для мониторинга доступности сетевых узлов в реальном времени. Написано на SwiftUI с поддержкой старых версий системы (вплоть до iOS 15.6), что позволяет использовать его даже на таких классических устройствах, как iPhone 7 Plus.

🎯 Предназначение
Приложение создано для системных администраторов, сетевых инженеров и продвинутых пользователей, которым нужно быстро проверить статус своих серверов, сайтов или IoT-устройств прямо с мобильного телефона.

✨ Основные возможности
Мониторинг в реальном времени: Проверка статуса хостов по протоколу ICMP (Ping).

Групповое управление: Запуск и остановка мониторинга всех добавленных узлов одной кнопкой.

Интуитивный интерфейс: Чистый дизайн на SwiftUI, адаптированный под темную тему.

Детальная статистика: Просмотр IP-адресов и описаний для каждого устройства.

Обратная совместимость: Полная поддержка iOS 15.0+, включая корректную работу на устройствах с кнопкой "Домой".

🛠 Технологический стек
Language: Swift

Framework: SwiftUI

Architecture: MVVM (Model-View-ViewModel)

Network: Основано на библиотеке SimplePing (Apple) с использованием Bridging Header для интеграции Objective-C кода.
